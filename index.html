Get Start

Welcome To Imagine Css Library !

Installation :

Step 1 : Head Over To your Project Folder And Open your terminal and type this text :
git clone https://github.com/ImagineNo1/imagine-css-library.git
Step 2 : Go to imagine-css-library folder and open the terminal and type this command :
npm install
Step 3 : in the terminal type this commad and wait untill the process finish
gulp
Step 4 : your css file is in this path :
imagine-css-library/css/index.css
Step 5 : Also your javascript bundle is in this path :
imagine-css-library/imagine/js/index.js

Step 6 :Use this two files into your project And its all done !

Offcanvas

Imagine Css library has its own offcanvas Component which you can see one of the right now by clicking on About Us in navbar And it has diffrent sizes and diffrent dirctions

How to use offcanvas ?

well to use offcanvas component you need to do following steps

.offcanvas-end : you need to use it for your div tag (This is right side )
.offcanvas-start : you need to use it for your div tag (This is left side)
.offcanvas-content : put your all stuff iniude the div with this class
.closeOffcanvas : You need to use this inside your ".offcanvas-content" to have close button
offcanvasDisplay() : In your ".offcanvas-start" or ".offcanvas-start" tag it is important to use and specefic id like offcanvasone and then on every element you want you need to use offcanvasDisplay(offcanvasone)

Example :

<div class = "offcanvas-end" id = "offcanvasOne">
   <div class="offcanvas-content" >
   <div class="closeOffcanvas"><div>
   Imagine Css Library
  </div>
</div>
<button onclick="offcanvasDisplay(offcanvasOne)"> Show offcanvas < /button>

Offcanvas have 2 directions and size variaty . for left direction use offcanvas-start and for right direction use offcanvas-end
for big size use offcanvas-end-lg and for small size use offcanvas-end-sm . The default value is medium .

Example :

<div class = "offcanvas-start-lg" id = "oocanvasTwo">
   <div class="offcanvas-content" >
   <div class="closeOffcanvas"><div>
   This is a left sided offcanvas and it has big size
  </div>
</div>
<button onclick="offcanvasDisplay(offcanvasTwo)"> Show offcanvas < /button>

This is a left sided offcanvas and it has big size
<div class = "offcanvas-end-sm" id ="offcanvasThree">
   <div class="offcanvas-content" >
   <div class="closeOffcanvas"><div>
   This is a right sided offcanvas and it has small size
  </div>
</div>
<button onclick="offcanvasDisplay(offcanvasThree)"> Show offcanvas < /button>

This is a right sided offcanvas and it has small size

You can Also use .close class on every html tag inside your offcanvas and when you click on that it close the offcanvas

GridSystems

Imagine Library has it own grid system which allows you to hav a fully responsive website in all sacales

Imagine grid system makes whole component to 12 columns , you can choose how many columns and element can have in every scale of screen

Note : Every line can have 12 columns if your elemts have more than 12 columns then the gonna go to next line

How We can use this grid system is super easy to learn . First of all you need to use .row class . Second We gonna use are .col classes inside it

Example :

<div class="row">
 <div class= "col-6">
   This Div Has 6 columns
 </div>
 <div class= "col-6">
   This Div Has 6 columns Too
 </div>
</div>
This Div Has 6 columns
This Div Has 6 columns

Imagine Grid System Have Some BreakPoints For screen sizes , when the screen size pass over them the rules are gonna change !

xs : ( Minimum width : 0 px )
sm : ( Minimum width : 480 px )
md : ( Minimum width : 720 px )
lg : ( Minimum width : 960 px )
xl : ( Minimum width : 1200 px )
xxl : ( Minimum width : 1440 px )

Note : For example if browser width passed from 480 px which is start of sm width then the classes with xs are not gonna worl

For using this brack points for making fully responsive website just add this brackpoints at the end off the class, watch the Example To learn .

Example :

<div class="row">
 <div class= "col-12-xs col-8-md col-5-lg">
   Lorem ipsum, dolor sit amet consectetur adipisicing elit.
 </div>
 <div class= "col-12-xs col-8-md col-5-lg">
   Lorem ipsum dolor, sit amet consectetur adipisicing elit.
 </div>
</div>
change your screen size to feel changes
change your screen size to feel changes

We have three classes for designing better this grid system which you can use with .row class :
justify-content-center : you can also use this names instead of center : (around , between , end , start)
align-items-center you can also use this names instead of center : (around , between , end , start)
gap-3 you can also use numbers from 1 to 5 instead of 3 , it just have diffrent gap you can choose between

Example :

<div class="row justify-content-center gap-2">
 <div class= "col-12-xs col-8-md col-5-lg">
   Lorem ipsum, dolor sit amet consectetur adipisicing elit.
 </div>
 <div class= "col-12-xs col-8-md col-5-lg">
   Lorem ipsum dolor, sit amet consectetur adipisicing elit.
 </div>
</div>
change your screen size to feel changes
change your screen size to feel changes
At The End of grid system , You can use this breack points at the end off all utilities which is in utility page like :
fs-md-xs and fs-xxl-md

Example :

<div class="fs-md-xs fs-xl-md">
  Look At This Text Change size
</div>
